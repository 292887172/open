  1. api 主要存放纯接口部分内容，例如：获取UI配置，保存邮寄地址等
  2. app 主要是页面相关全部路由表和具体实现，包括管理后台和开发平台
  3. base 主要是基础通用部分内容，包括数据库连接器，静态固定变量定义，标准默认模板等
  4. common 通用部分内容，数据库具体读写操作封装， 异步任务，发送邮件短信服务，生成lua脚本文件替换
  5. conf 配置文件存放路径
  6. doc 项目相关文档存放路径
  7. idl 服务配置存放路径，例如：设备在线状态
  8. log 项目运行日志
  9. model 数据库模型文件
  10. static 项目使用的静态资源文件路径，包括angularjs文件，bootstrap文件,markdown编辑器文件，layui框架文件，sdk文件存放，ejs模板存放
  11. templates 前端页面模板存放，主要是django前端模板存放路径，包括管理后台页面
  12. util 通用工具类封装，邮件发送，短信发送等
