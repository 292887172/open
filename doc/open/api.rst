api文档
=========


全局返回码说明
-------------------
=============    ========================================================
返回码                  说明
=============    ========================================================
-1                      系统繁忙,检查参数后再试
0                       请求成功
41001                   缺少设备授权key
40004                   app_key无效
=============    ========================================================


获取待配网设备基础信息接口
--------------------------

* 接口地址：http://test.open.53iq.com/product/verify
* HTTP方式：POST (通过接口访问)

输入参数
^^^^^^^^
=============    ========================================================
参数                  说明
=============    ========================================================
key                  请求的App中的授权key
=============    ========================================================

输出参数
^^^^^^^^
::

        {
            "code": 0,
            "data": "http://test.open.53iq.com/static/file/m23cYZki.zip",
            "msg": "success"
        }
        """
        code不为0,msg为具体错误信息
        """

扫码推送zip文件
--------------------------

* 接口地址：http://test.open.53iq.com/product/wx_code
* HTTP方式：POST(通过接口访问)

输入参数
^^^^^^^^
=============    ========================================================
参数                  说明
=============    ========================================================
key                  产品标识符
id                   设备编号
=============    ========================================================

输出参数
^^^^^^^^
::

        {
            "code": 0,
            "data": {
                "total": 1
            }
        }
        或者
        {
            "code": 42001,
            "msg": "push message failed"
        }
        """
        code不为0,msg为具体错误信息
        """
